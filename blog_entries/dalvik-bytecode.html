Dalvik Bytecode

February 19, 2016 by Edward

Recently, I spoke with the founder of PNF software, a company responsible for creating the JEB decompiler for Android applications. Dalkvik, compiled from Java, is a space-saving alternative to Java Bytecode. Android uses a Linux-based kernel, but applications run in Dalvik/Java-based sandboxes.
