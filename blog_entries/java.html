Original motivation for Java was not the Internet, but rather need for platform-independent language for embedded systems with a variety of CPUs/controllers. Initially called "Oak," but later renamed to "Java." Created by 5 guys in 1991 at Sun Microsystems. The Internet increased the importance of code portability. Syntax inspired by C and OOP features inspired by C++. Java is not "C++ adapted for the Internet" ... It has several fundamental underlying differences and themes from C++. Java was not intended to replace C++. It addresses a totally different scenario than C++. C# was designed by Microsoft for the .NET framework and it is very similar to Java in that they both have C++ based syntax, allow distributed programming, follow an object-based paradigm. However, C# and Java are optimized for different scenarios. Applet = "Special kind of Java program that is design to be transmitted over the Internet and automatically executed by a Java-compatible Web browser." "Downloaded on demand" like JS today.
