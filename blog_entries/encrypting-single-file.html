Encrypting a Single File

February 8, 2016 by Edward

Sometimes, you want to encrypt a single file or group of files such that a password can decrypt them. There are a few options:

  • Encrypted and password-protected PDF. I think it uses RC4 by default.
  • Use the Mac Disk Utility
  • Use openssl
  • Probably many more options

Here I will explain the openssl option. I am borrowing/copying from Dennis on SuperUser.com/StackExhange, originally posted here. Dennis copied the commands from here.

"opensll comes pre-installed on Mac OS X." Here are the relevant commands:

# encrypt file.txt to file.enc using 256-bit AES in CBC mode
openssl enc -aes-256-cbc -salt -in file.txt -out file.enc

# the same, only the output is base64 encoded for, e.g., e-mail
openssl enc -aes-256-cbc -a -salt -in file.txt -out file.enc

# decrypt binary file.enc
openssl enc -d -aes-256-cbc -in file.enc -out file.txt

# decrypt base64-encoded version
openssl enc -d -aes-256-cbc -a -in file.enc -out file.txt

"These commands use 256-bit AES ecryption with Cipher Block Chaining (CBC), which is about as secure as it gets right now."
