Parsing Date/Time Formats in Python Quickly

February 8, 2016 by Edward

For a recent coding project, I needed to compare dates and times that were in several different formats, such as:

  • "2015-03-31T21:59:12.967-04:00"
  • "Sun Apr 6 00:51:06 2014 +0100"

My strategy was to convert all the input formats into a common uniform format, such as the Unix timestamp, to enable easy comparisons. However, writing the conversion code for each format was tedious. I initially used the following code:

     import datetime
     import time
     month = git_time_string.split()[1]
     day = git_time_string.split()[2]
     year = git_time_string.split()[-2]
     hour = git_time_string.split()[3].split(':')[0]
     min = git_time_string.split()[3].split(':')[1]
     sec = git_time_string.split()[3].split(':')[2]
     date_time_string = "%s %s %s %s %s %s" % (month, day, year, hour, min, sec)
     date_time = datetime.datetime.strptime(date_time_string, '%b %d %Y %H %M %S')
     unix_time = time.mktime(date_time.timetuple())
